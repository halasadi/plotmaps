# plotmaps

An R-package for plotting the output of the MAPS software (https://github.com/halasadi/MAPS). 

# Installation

```
git clone https://github.com/halasadi/plotmaps.git
install.packages("devtools")
library(devtools)
install(path-to-plotmaps/plotmaps)
```
